## Summary
This website is a simplified version of job search service. It's a CRUD application.
User can read, add, update and delete vacancies and candidates. User is unlimited in adding
new items of vacancies or candidates. User can use this website as a guest or 
being logged in after signing up.

## Tech Stack:
- Java 17
- Spring Core 5
- Spring boot 2
- Hibernate 5
- PostrgreSQL 15
- SLF4j 1
- Liquibase 4
- Mockito 4
- Thymeleaf 3
- Bootstrap 5
- HTML, CSS

## Environment
JDK 17, IntelliJ IDEA CE 2021.3.3, Maven 3.8.1, PostrgreSQL 15
